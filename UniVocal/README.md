# UniVocal: Unified Speech-Singing Code-Switching Synthesis
> **🎉 Accepted to the Main Conference of ACL 2026!**
>
> **🚧 Coming Soon**
> 
> This repository is currently under construction. We are actively organizing and cleaning up the codebase for public release.

This is the official repository for **UniVocal**, a unified framework that implicitly infers vocal modes from text context to pioneer Speech-Singing Code-Switching (SCS) Synthesis.

## 📦 What to Expect
We will be releasing the complete pipeline in the near future. The upcoming release will include:

- [ ] **Inference Code:** Scripts for generating Speech, Singing, and seamless Code-Switching audio.
- [ ] **Pre-trained Checkpoints:** Ready-to-use model weights.
- [ ] **Training Code:** Scripts to reproduce our two-stage curriculum learning strategy.
- [ ] **Data Processing Scripts:** The complete pipeline for dataset construction.
